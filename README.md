## Trilha Python DIO
Repositório com exercícios e projetos desenvolvidos durante a Trilha de Python da DIO (Digital Innovation One).
---
## Sobre o projeto
A Trilha Python DIO é um projeto de estudo focado no aprendizado progressivo da linguagem Python, desde os fundamentos até conceitos mais avançados como Programação Orientada a Objetos, manipulação de arquivos e desenvolvimento de APIs.
O objetivo deste repositório é registrar a evolução no aprendizado e consolidar conhecimentos na área de desenvolvimento backend.
---
Conteúdos abordados
Fundamentos da linguagem Python
Variáveis, tipos de dados e operadores
Estruturas condicionais e de repetição
Listas, tuplas, conjuntos e dicionários
Funções e modularização
Programação Orientada a Objetos (POO)
Tratamento de erros e exceções
Manipulação de arquivos
Criação de APIs com Flask e FastAPI
Integração com banco de dados

Como utilizar o projeto
Clonar o repositório pelo GitHub
Acessar a pasta do projeto
Executar os arquivos Python no terminal ou VS Code
Testar APIs utilizando navegador ou Postman (quando aplicável)

Tecnologias utilizadas
Python 3
Flask
FastAPI
SQLite / Banco de Dados SQL
Git e GitHub
Visual Studio Code
